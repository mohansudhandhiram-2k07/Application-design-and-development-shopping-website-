TechStore is a simple, static website built with HTML, Tailwind CSS, and vanilla JavaScript that allows users to compare the specifications of different laptops side-by-side.

All laptop data is hardcoded directly into the compare.html file. The site provides a clean interface to select two devices and view their CPU, RAM, storage, and price in a simple table.
